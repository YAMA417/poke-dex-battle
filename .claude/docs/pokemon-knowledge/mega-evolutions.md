# メガシンカ 全データ

## メガシンカの仕様

- 1バトルにつき1匹のみメガシンカ可能
- 対応するメガストーン（◯◯ナイト）の装備が必要
- ターン消費なしで発動（技選択と同時）
- バトル終了まで継続（任意解除不可）
- ひんし時は解除される
- 手持ち交換しても維持
- メガシンカターンは**メガシンカ前**の素早さで行動順が決まる
- メガストーンはトリック/はたきおとしで奪えない

## メガストーンとアイテムカテゴリ

poke-dex-battle の `item-categories.ts` では、メガストーンは `mega: ['mega-stone', 'orb']` カテゴリに分類。
ポケモンの `fixedItemId` でメガシンカ石を参照する設計。
