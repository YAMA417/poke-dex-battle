# ポケモンチャンピオンズ 固有の変更点

ポケモンチャンピオンズ（2026年4月8日配信）における従来作品からの変更点。

## ゲーム概要

- バトル特化型。Nintendo Switch / Switch 2 / Android / iOS
- 基本プレイ無料。マイクロトランザクションあり（VP購入不可、バトルパスあり）
- 最終進化ポケモンのみ登場（一部例外あり）
- ポケモンHOME連携可能（ゲーム内解禁済みポケモンのみ）
- シングルバトル / ダブルバトル対応

---

## 能力値システム

### 個体値の廃止
全ポケモンの個体値は31相当で統一。個体値による個体差は存在しない。

### 能力ポイント（旧努力値に相当）

| 項目 | 値 |
|------|-----|
| 合計上限 | **66** |
| 1ステータス上限 | **32** |
| 1ポイントの効果 | 実数値+1（直接加算） |

旧作のEV（努力値）と異なり、`EV÷4` ではなく **1ポイント=実数値1** の直接加算。

### 実数値計算式

**HP:**
```
実数値 = 種族値 + 75 + 能力ポイント
```

**HP以外（攻撃・防御・特攻・特防・素早さ）:**
```
実数値 = (種族値 + 20 + 能力ポイント) × 性格補正
```
※ 小数点以下切り捨て

### 性格補正
- 上昇: ×1.1
- 下降: ×0.9
- 無補正: ×1.0

### レベル
ランクバトルはLv50固定。レベルの概念自体が簡略化されている。

## PP変更

全技のPPが再設定された。PP増加手段（ポイントアップ/ポイントマックス）は存在しない。

| 旧PP | 新PP |
|------|------|
| 5 | **8** |
| 10 | **12** |
| 15 | **16** |
| 20以上 | **20** |

### 例外
- **まもる**: 8PP（旧16から半減）

## 新特性（4種）

### メガソーラー（Mega Sol）
- **効果**: この特性を持つポケモンが技を使うとき、天気が「にほんばれ」状態と同じ効果を受ける
- **所持**: メガメガニウム
- **備考**: ソーラービーム溜めなし、こうごうせい回復量増加など。実際に天候を変えるわけではなく、技使用時に晴れの恩恵を受ける

### ドラゴンスキン（Dragonize）
- **効果**: 自分のノーマルタイプの技がドラゴンタイプになり、威力が1.2倍になる
- **所持**: メガオーダイル
- **備考**: スキン系特性（フリーズスキン等）と同系統

### かんつうドリル（Piercing Drill）
- **効果**: 接触技がまもる・みきり等を貫通し、25%のダメージを与える
- **所持**: メガドリュウズ
- **備考**: ふかしのこぶし（弱体化後）と同等の効果を持つ新特性

### とびだすハバネロ（Spicy Spray）
- **効果**: 攻撃技でダメージを受けると、攻撃してきた相手をやけど状態にする
- **所持**: メガスコヴィラン
- **備考**: 接触・非接触を問わず発動。ほのおタイプの相手にはやけどは効かない

## 技・特性の調整

### 特性の変更
- **ふかしのこぶし（Unseen Fist）**: まもり貫通時のダメージが100%→**25%**に弱体化

### いかく（Intimidate）の変更
- ダブルバトルで相手2体の攻撃を**同時に**下げるように変更（従来は順次処理）

### 技の強化
| 技名 | 変更内容 |
|------|----------|
| Gのちから | 威力 80→**90** |
| クラブハンマー | 命中率 90→**95** |
| どくのいと | 素早さ低下 1段階→**2段階** |
| トラバサミ | タイプ くさ→**はがね** |
| ポルターガイスト | メガストーンに対して成功するように変更 |

### 技の弱体化
| 技名 | 変更内容 |
|------|----------|
| ムーンフォース | 特攻低下確率 30%→**10%** |
| アイアンヘッド | ひるみ確率 30%→**20%** |
| しおづけ | ダメージ 最大HP1/8→**1/16**（弱点相手: 1/4→**1/8**） |
| はめつのひかり | 反動ダメージ 1/3→**1/2** |
| シャドーボール | 特防低下確率 20%→**10%** |
| フリーズドライ | こおり状態を**付与しなくなった** |

### 技の没収
| ポケモン | 没収された技 |
|----------|-------------|
| ガオガエン | はたきおとす、とんぼがえり |
| ゲンガー | アンコール |
| カイリュー | アンコール |
| ブリジュラス | ボディプレス |

※ 過去に「ガルーラ × グロウパンチ」「ロトム × すてゼリフ／パラボラチャージ」も没収リストに含めていたが、SVのlearnsetを確認したところ元々これらは覚えない（メガガルーラのグロウパンチ等、メガシンカ前提技は除外）。実態と齟齬があったため除外した。

### 技の追加
| ポケモン | 追加された技 |
|----------|-------------|
| メガカメックス | からをやぶる |
| ギルガルド | ポルターガイスト |
| リザードン | はねやすめ（再習得） |
| ハッサム | はねやすめ（再習得） |
| メガミミロップ | インファイト、マッハパンチ、とんぼがえり、トリプルアクセル |

### 技カテゴリの追加
- **シャドークロー / ドラゴンクロー**: 「スライス技」カテゴリに追加（きれあじで威力1.5倍の対象に）

### 技情報の表示強化
- 追加効果の発生確率を明示表示
- 接触技、音技、弾系技などのカテゴリが技説明に表示
- 先制技の優先度ブラケット表示

## 状態異常の仕様変更

| 状態異常 | 旧仕様 | 新仕様 |
|----------|--------|--------|
| まひ | 25%で行動不可 | **12.5%**で行動不可 |
| こおり | 永久（毎ターン20%で自然解凍） | **3ターン以内に必ず解凍** |
| ねむり（技による） | 2ターン | **3ターン** |

## TOD（Time Over Decision）の変更

- 旧仕様: 試合時間終了時、残りHP割合で勝敗判定
- 新仕様: 試合時間終了時は**引き分け**

---

## 育成・カスタマイズシステム

VP（Victory Points）を使ってポケモンをカスタマイズ：

| 項目 | コスト |
|------|--------|
| 能力ポイント調整 | 2 VP / 1ポイント |
| 技変更 | 100〜250 VP / 1技 |
| 性格変更 | 200〜500 VP |
| 特性変更 | 400〜500 VP |

※ VPはバトル・ミッション報酬で獲得。課金では購入不可。

## 未実装アイテム

以下のアイテムはチャンピオンズに実装されていない:
- **どくどくだま**: グライオン（ポイズンヒール）の能動発動が不可
- **かえんだま**: ミロカロス（ふしぎなうろこ）、ローブシン（こんじょう）の能動発動が不可

## 内定ポケモン

2026年4月8日時点で **218体**（メガシンカ60体を含めると計278体）。
詳細リストは `available-pokemon.md` を参照。
- 最終進化のみが基本（一部例外: ピカチュウ、エモンガ等）
- ポケモンHOMEから転送可能（解禁済みのもののみ）
- ロスターランチ: 22時間ごとに10体がランダム表示、VPで勧誘可能

## レギュレーション

### レギュレーションM-A（2026/4/8〜6/17）
- シングルバトル: 選出3体 / ダブルバトル: 選出4体
- メガシンカ: 使用可能（1バトル1回）
- ダイマックス / テラスタル / Z技: **使用不可**
- 使用可能: 213体（フォーム含む）
- 伝説ポケモン: 使用不可
- 幻ポケモン: 使用不可
- 対戦時間: 総合20分、持ち時間7分、選択45秒、選出90秒

- 能力ランク倍率を数値で表示（例: 1段階下降 → 「0.67x」、2段階上昇 → 「2.0x」）
- 4倍弱点時「効果は ちょうバツグンだ！」表示
- 技の効果範囲・対象がパーティ全体に対して表示
