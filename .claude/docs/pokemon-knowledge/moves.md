# 技の仕様・分類体系

## 技のカテゴリ

技は3つのカテゴリに分類される：

| カテゴリ | 説明 |
|----------|------|
| 物理 | 攻撃→防御で計算 |
| 特殊 | 特攻→特防で計算 |
| 変化 | ダメージなし。状態変化・能力変化等 |

---

## 技フラグ

技にはダメージ計算や特性との相互作用に影響するフラグがある。
poke-dex-battle の moves テーブルのカラムに対応。

| フラグ | 説明 | 関連特性 |
|--------|------|---------|
| isContact | 接触技 | かたいツメ(×1.3)、もふもふ(被ダメ半減) |
| isPunch | パンチ系技 | てつのこぶし(×1.5) |
| isBite | キバ系技 | がんじょうあご(×1.5) |
| isAura | 波動・はどう系技 | メガランチャー(×1.5) |
| isRecoil | 反動技 | すてみ(×1.2)、いしあたま(反動無効) |
| isSlicing | スライス・切断技 | きれあじ(×1.5) |
| isSound | 音技 | パンクロック(×1.3)、みがわり貫通 |
| isBullet | 弾・爆弾系技 | ぼうだん(無効化) |
| isWind | 風技 | かぜのり(攻撃+1、ダメージ無効) |

### チャンピオンズでの変更
- **シャドークロー / ドラゴンクロー** が `isSlicing` に追加

## 優先度

技には優先度（priority）があり、同優先度内で素早さ順に行動する。

| 優先度 | 主な技 |
|--------|--------|
| +5 | てだすけ |
| +4 | まもる、みきり、キングシールド、ニードルガード |
| +3 | ねこだまし、ファストガード、ワイドガード |
| +2 | しんそく、フェイント |
| +1 | アクアジェット、マッハパンチ、バレットパンチ、かげうち、こおりのつぶて、でんこうせっか、ふいうち |
| 0 | 通常の技 |
| -1 | — |
| -2 | — |
| -3 | きあいパンチ |
| -5 | — |
| -6 | ほえる、ふきとばし |
| -7 | トリックルーム |

## 連続技

複数回ヒットする技。各ヒットで個別にダメージ計算。

| ヒット数 | 主な技 |
|----------|--------|
| 2回固定 | ダブルアタック、にどげり |
| 3回固定 | トリプルアクセル、トリプルキック（ヒットごとに威力+20） |
| 5回固定 | つららばり（確定5回）、ミサイルばり（確定5回）…ではなく2-5回 |
| 2-5回 | つららばり、ロックブラスト、ミサイルばり、タネマシンガン |
| おやこあい | 2回（2発目は威力1/4） |

2-5回ヒットの確率：
| 回数 | 確率 |
|------|------|
| 2回 | 35% |
| 3回 | 35% |
| 4回 | 15% |
| 5回 | 15% |

スキルリンク持ちは常に5回。

## 固定ダメージ技

通常のダメージ計算式を使わない技：

| 技 | ダメージ |
|----|---------|
| ちきゅうなげ / ナイトヘッド | レベルと同じ（=50） |
| りゅうのいかり | 40固定 |
| ソニックブーム | 20固定 |
| いのちがけ | 自分のHP分 |
| がむしゃら | 相手HP-自分HP分 |
| カウンター | 受けた物理ダメージの2倍 |
| ミラーコート | 受けた特殊ダメージの2倍 |

## 自分の能力で威力が変わる技

| 技 | 威力計算 |
|----|---------|
| ジャイロボール | 150×(相手素早さ÷自分素早さ)、最大150 |
| けたぐり / くさむすび | 相手の体重で威力変動（10kg→20、200kg+→120） |
| ボディプレス | 防御の実数値で物理ダメージを計算（攻撃の代わり） |
| イカサマ | 相手の攻撃実数値でダメージ計算 |
| テラバースト | 攻撃>特攻なら物理、そうでなければ特殊 |

## 主要な変化技（対戦頻出）

| 技 | 効果 |
|----|------|
| まもる | そのターンの技を完全に防ぐ。連続使用で成功率低下 |
| トリックルーム | 5ターン、素早さが遅い順に行動 |
| おいかぜ | 4ターン、味方全体の素早さ2倍 |
| てだすけ | 味方1体の技の威力1.5倍（そのターンのみ） |
| このゆびとまれ | そのターン、相手の単体技を自分に引き付ける |
| いかりのこな | このゆびとまれと同効果（くさタイプには無効） |
| サイドチェンジ | 味方と位置を入れ替える |
| トリック / すりかえ | 相手と持ち物を交換 |
| ちょうはつ | 3ターン変化技を使えなくする |

## チャンピオンズでの技変更

### PP変更

全技のPPが再設定された。ポイントアップ/ポイントマックスは存在しない。

| 旧PP | 新PP |
|------|------|
| 5 | **8** |
| 10 | **12** |
| 15 | **16** |
| 20以上 | **20** |

**例外**: まもる = 8PP（旧16から半減）

### 技の強化

| 技名 | 変更内容 |
|------|----------|
| Gのちから | 威力 80→**90** |
| クラブハンマー | 命中率 90→**95** |
| どくのいと | 素早さ低下 1段階→**2段階** |
| トラバサミ | タイプ くさ→**はがね** |
| ポルターガイスト | メガストーンに対して成功するように変更 |
