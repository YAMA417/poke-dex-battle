# 対戦で重要な特性一覧

ダメージ計算・バトルメカニクスに影響する主要特性。
poke-dex-battle の `damageEffect` 設計の参考に使う。

## ダメージ計算に直接影響する特性

### 攻撃力倍化系

| 特性 | 効果 | 補正値(4096基準) |
|------|------|-----------------|
| ちからもち | 物理技の攻撃力2倍 | 8192 |
| ヨガパワー | 物理技の攻撃力2倍 | 8192 |
| すいほう | みず技の威力2倍 + ほのお技被ダメ半減 | 8192(みず技時) |
| はりこみ | 交代で出てきた相手への攻撃力2倍 | 8192 |

### 防御力倍化系

| 特性 | 効果 | 補正値 |
|------|------|--------|
| ファーコート | 物理技の被ダメージ半減 | 2048 |
| ふしぎなうろこ | 状態異常時に防御1.5倍 | 6144 |
| アイスフェイス | 物理技を1回無効（アイスフォルム時） | — |

### 威力補正系（×1.5 = 6144）

| 特性 | 条件 |
|------|------|
| テクニシャン | 威力60以下の技 |
| がんじょうあご | キバ系技（かみくだく、こおりのキバ等） |
| メガランチャー | 波動・はどう系技（はどうだん、みずのはどう等） |
| てつのこぶし | パンチ系技（れいとうパンチ、かみなりパンチ等） |
| すてみ | 反動技（すてみタックル、フレアドライブ等） |
| きれあじ | スライス・切断技（リーフブレード、シザークロス、シャドークロー等） |

### 威力補正系（×1.3 = 5325）

| 特性 | 条件 |
|------|------|
| かたいツメ | 接触技 |
| パンクロック | 音技 |
| とうそうしん | 同性の相手への攻撃（異性には×0.75） |

### 威力補正系（×1.2 = 4915）

| 特性 | 条件 |
|------|------|
| スナイパー | 急所時（×1.5→×2.25になる） |
| いろめがね | 「いまひとつ」の技のダメージ2倍（実質等倍に） |

### STAB補正系

| 特性 | 効果 |
|------|------|
| てきおうりょく | タイプ一致補正が1.5→2.0に |

## バトル開始時・交代時に発動する特性

| 特性 | 効果 |
|------|------|
| いかく | 相手全体の攻撃1段階下降（チャンピオンズでは同時処理） |
| ダウンロード | 相手の防御<特防なら特攻+1、そうでなければ攻撃+1 |
| かわりもの | 相手に変身（ステータス・技をコピー） |
| トレース | 相手の特性をコピー |
| おみとおし | 相手の持ち物を確認 |

## その他の重要な対戦特性

| 特性 | 効果 |
|------|------|
| かたやぶり | 相手の特性を無視して攻撃 |
| へんげんじざい | 技を出す前に自分のタイプがその技のタイプに変化（1回のみ） |
| ばけのかわ | 1回だけ攻撃を無効化（HP1/8のダメージは受ける） |
| がんじょう | HP満タン時にひんしになる攻撃を耐える（HP1残り） |
| マジックガード | 技以外のダメージを受けない |
| さいせいりょく | 交代時にHP1/3回復 |
| おやこあい | 技が2回連続で当たる（2発目は威力1/4） |
| こんじょう | 状態異常時に攻撃1.5倍、やけどの物理低下無効 |
| メガソーラー（新） | 技使用時に晴れ状態の効果を受ける |
