# ポケモンバトル 基本メカニクス

## ダメージ計算式

### 基本式

```
ダメージ = ((レベル×2÷5+2) × 威力 × A÷D ÷50 +2) × 各種補正
```

- A = 物理技なら「こうげき」、特殊技なら「とくこう」
- D = 物理技なら「ぼうぎょ」、特殊技なら「とくぼう」
- 各ステップで小数点以下切り捨て
- レベルは50固定（ランクバトル）

### 乱数

最終ダメージに 0.85〜1.00 の16段階乱数を掛ける（切り捨て）。

### 補正の適用順序

ダメージ計算では「五捨五超入」（4096基準）が使われる。
補正値は 4096 を基準とし、`ダメージ × 補正値 / 4096` で切り捨て。

適用順序（概略）：
1. 複数対象補正
2. おやこあい補正
3. 天候補正
4. 急所補正
5. 乱数
6. タイプ一致（STAB）
7. タイプ相性
8. やけど補正
9. 最終ダメージ補正（壁、特性、アイテム等の乗算）

## タイプ相性

### 倍率

| 相性 | 倍率 |
|------|------|
| 効果抜群 | ×2.0 |
| いまひとつ | ×0.5 |
| 効果なし | ×0 |
| 通常 | ×1.0 |

ダブルタイプの場合は乗算（例: 4倍弱点 = 2.0 × 2.0 = 4.0）。

## ランク補正（能力ランク）

能力変化は -6 〜 +6 の13段階。

| ランク | 倍率 |
|--------|------|
| -6 | 2/8 = 0.25 |
| -5 | 2/7 ≈ 0.286 |
| -4 | 2/6 ≈ 0.333 |
| -3 | 2/5 = 0.4 |
| -2 | 2/4 = 0.5 |
| -1 | 2/3 ≈ 0.667 |
| 0 | 2/2 = 1.0 |
| +1 | 3/2 = 1.5 |
| +2 | 4/2 = 2.0 |
| +3 | 5/2 = 2.5 |
| +4 | 6/2 = 3.0 |
| +5 | 7/2 = 3.5 |
| +6 | 8/2 = 4.0 |

計算式: ランク≥0 なら `(2+ランク)/2`、ランク<0 なら `2/(2-ランク)`

命中・回避のランク補正も同じ計算式。

## 急所

### 急所ランク

| ランク | 確率 |
|--------|------|
| 0 | 1/24 |
| +1 | 1/8 |
| +2 | 1/2 |
| +3以上 | 確定 |

### 急所ダメージ

- 倍率: **×1.5**（第6世代以降）
- 攻撃側のランク下降を無視
- 防御側のランク上昇を無視
- リフレクター/ひかりのかべを無視

## 天候

4種類の天候が存在。5ターン継続（岩系アイテムで8ターン）。

| 天候 | 効果 |
|------|------|
| にほんばれ（晴れ） | ほのお技×1.5、みず技×0.5、ソーラービーム溜めなし、かみなり命中50% |
| あめ | みず技×1.5、ほのお技×0.5、かみなり必中、ソーラービーム威力半減 |
| すなあらし（砂嵐） | いわタイプの特防×1.5、毎ターン最大HPの1/16ダメージ（岩/地/鋼/砂系特性除く） |
| ゆき（雪） | こおりタイプの防御×1.5 |

---

## フィールド

4種類のフィールド。5ターン継続（対応アイテムで8ターン）。地面にいるポケモンに影響。

| フィールド | 効果 |
|------------|------|
| エレキフィールド | でんき技×1.3、ねむり無効 |
| グラスフィールド | くさ技×1.3、毎ターンHP1/16回復、じしん/じならし/マグニチュード威力半減 |
| サイコフィールド | エスパー技×1.3、先制技無効 |
| ミストフィールド | ドラゴン技×0.5、状態異常無効 |

---

## タイプ一致ボーナス（STAB）

- 通常: **×1.5**
- てきおうりょく持ち: **×2.0**

自分のタイプと一致するタイプの技を使うとき適用。

---

## 複数対象補正（ダブルバトル）

相手2体に当たる技: **×0.75**

味方を巻き込む技（じしん等）も同様。
対象が1体しかいない場合は補正なし。

---

## やけど補正

物理技のダメージ: **×0.5**
（からげんき、特性「こんじょう」持ちは除く）

---

## 壁補正

| 壁 | 効果（シングル） | 効果（ダブル） |
|----|-----------------|---------------|
| リフレクター | 物理ダメージ×0.5 | 物理ダメージ×0.667 |
| ひかりのかべ | 特殊ダメージ×0.5 | 特殊ダメージ×0.667 |
| オーロラベール | 物理・特殊×0.5 | 物理・特殊×0.667 |
