# ポケモンダブルバトル支援アプリ 開発スケジュール

## 概要

| Phase | 内容 | 備考 |
|-------|------|------|
| Phase 1-1 | ダメージ計算機能 | 最初のリリース（フロントのみ） |
| Phase 1-2 | パーティ登録機能 | フロントのみ |
| Phase 1-3 | 対戦履歴管理 | フロントのみ |
| Phase 2 | 自前バックエンド + 認証 | Hono + PostgreSQL + JWT |
| Phase 3 | スマホアプリ化（Expo） | Web版完成後 |

**開発順序**: Web版（フロントのみ）→ バックエンド追加 → スマホアプリ化

## 将来の拡張・移行

### Supabaseへの移行（必要に応じて）
- PostgreSQL互換のため、スキーマはそのまま移行可能
- 認証をSupabase Authに置き換え
- Prismaの接続先を変更するだけで移行可能

### その他の検討事項
- パーティ構築支援（タイプ相性チェック、弱点分析）
- パーティ共有機能（QRコード、URL）
- コミュニティ機能（パーティ投稿・評価）
- プッシュ通知（大会情報など）
- 多言語対応
- ポケモンチャンピオンズ対応
